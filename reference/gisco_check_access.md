# Check access to GISCO API

Check if **R** has access to resources at
<https://gisco-services.ec.europa.eu/distribution/v2/>.

## Usage

``` r
gisco_check_access()
```

## Value

a logical.

## See also

Other helper:
[`gisco_attributions()`](https://ropengov.github.io/giscoR/reference/gisco_attributions.md)

## Examples

``` r
gisco_check_access()
#> [1] TRUE
```
